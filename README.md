# Lovi Care - Smart Skin Care Website Replica

An exact replica of the Lovi Care website (https://lovi.care/) - an AI-powered smart skin care platform.

## Project Overview

**Original Source:** https://lovi.care/  
**Project Type:** Static Website Replica  
**Built with:** HTML5, CSS3, JavaScript (Framer-generated)

## Description

Lovi is a science-backed AI-Cosmetologist platform that helps users:
- Set skincare goals and build best-fit skincare routines
- Use face scanner technology
- Check cosmetics safety
- Get personalized AI-powered skincare recommendations
